git commit 시 이전 로그의 메세지를 붙여 넣을 수 있다.  


## package {#package}

```elisp
(package! git-msg-prefix)
```


## config {#config}

```elisp
(use-package! git-msg-prefix
  ;; :hook
  ;; (git-commit-mode . git-msg-prefix)
  :custom
  (git-msg-prefix-log-flags " --since='1 week ago' ")
  (git-msg-prefix-input-method 'helm-comp-read))
```
