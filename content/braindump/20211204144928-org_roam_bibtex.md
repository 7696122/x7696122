## package {#package}

```elisp
(package! org-roam-bibtex)
```


## config {#config}

```elisp
(use-package! org-roam-bibtex
  :after org-roam
  :config
  (require 'org-ref)
  (add-to-list 'org-roam-capture-templates
               '("r" "bibliography reference" plain "%?"
                 :target
                 (file+head "references/${citekey}.org" "#+title: ${title}\n")
                 :unnarrowed t)))
```
