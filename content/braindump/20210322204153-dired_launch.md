## package {#package}

```elisp
(package! dired-launch)
```


## config {#config}

```elisp
(use-package! dired-launch
  :hook
  (dired-mode . dired-launch-mode)
  :config
  (setq dired-launch-default-launcher
        (or (and IS-MAC '("open"))
            (and IS-LINUX '("xdg-open"))))
  (dired-launch-enable))
```
